# Мой блог
## Описание проекта
### Цель проекта

Мой блог - это многопользовательская платформа для ведения личного блога с возможностью  
делиться полезной информацией с другими пользователями.
Позволяет изучить и оценить статьи других пользователей, а так же  
оставлять и получать обратную связь от читателей.

### Функциональность

- **Авторизация** - Вход в систему через GitHub OAuth
- **Посты** - Просмотр, редактирование, создание, удаление статей
- **Лайки** - Возможность ставить и снимать лайки на статьях
- **Комментарии** - Возможность добавления комментариев к статье
- **Фильтрация** - Возможность фильтровать статьи и комментарии по дате и авторам
- **Сортировка** - Сортировка по дате, лайкам, количеству комментариев и названию


## Технологии
git - Система контроля версий   
Docker - Платформа для создания и запуска образа  
uv - Пакетный менеджер для установки зависимостей

#### Список библиотек:
- **django** - основной фреймворк для создания приложения  
- **djangorestframework** - расширение Django для создания API  
- **django-filter** - библиотека для выборки данных  
- **drf-spectacular** - библиотека для генерации схем   
- **psycopg2-binary** - возможность работы с реляционной базой postgres  
- **requests** - библиотека для выполнения запросов ко внешним источникам  
- **django-cors-headers** - библиотека для управления заголовками запросов  
- **dj-database-url** - библиотека для настройки конфигурации подключения к БД  
- **django-ckeditor** - редактор текстовых полей  
- **uvicorn** - сервер для запуска приложений  
- **whitenoise** - обработка статичных файлов  
- **python-dotenv** - библиотека для работы с переменными окружения  


## Установка
Для запуска приложения потребуются установленный git и docker  
Распаковать файл .env.zip и положить извалеченный файл в корень проекта

Команды для запуска:  
git clone https://github.com/V-Filippov1994/MyBlog.git  
docker-compose --build  
docker-compose run --rm backend python manage.py migrate  
docker-compose up (-d для отключения логов)  

Выключение контейнеров  
docker-compose down --remove-orphans  

Создание административной учетной записи  
- docker-compose run --rm backend python manage.py createsuperuser
